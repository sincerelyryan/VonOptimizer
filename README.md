# VonOptimizer
A C# GUI Optimizer that debloats/tweaks Windows.

Tweaks from RisxnTweaks - https://risxntweaks.com/

- **CuoreUI** - UI Controls
- **Guna2UI** - UI Controls 
- **Costura.Fody** - DLL Embed (Allows you to just have an .exe file)


# Features

- Restore Point Creator
- System Configuration Debloat
- Clear Temp Files
- Disable Telemetry
- Block Updates
- CTT Power Plan
- IO Tweaks
- Windows Settings
- Uninstall Useless Apps
- Disable Processes
- Disable Printer
- MenuKillTime
- CPU Tweaks
- Reduce Mem usage
- Disable Limiting
- Network Reg settings
